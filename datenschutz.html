2. Welche Daten werden verarbeitet?

2.1. Technisch notwendige Daten

Beim Aufruf der Webseite verarbeitet der Hosting-Dienst Supabase automatisch:

  • anonymisierte IP-Adresse
  • Zeitpunkt des Zugriffs
  • Browsertyp / Betriebssystem
  • Abgerufene Inhalte (z. B. Kategorien, Einträge)

Diese Daten sind technisch erforderlich, um die Seite auszuliefern. Sie werden nicht zu Analysezwecken genutzt.

2.2. Keine Cookies, kein Tracking

MarketShield setzt keine Cookies, kein Tracking und keine Marketing-Pixel ein.

2.3. KI-Generierung von Inhalten

Einige Inhalte werden über externe KI-Systeme erstellt (z. B. DeepSeek). Dabei werden ausschließlich sachliche Begriffe übermittelt, niemals persönliche Daten.

3. Weitergabe von Daten

Es erfolgt keine Weitergabe personenbezogener Daten an Dritte. Supabase agiert ausschließlich als technischer Auftragsverarbeiter.

4. Rechte der Nutzer

Da MarketShield keine personenbezogenen Daten speichert, ergeben sich keine Auskunfts- oder Löschansprüche.

5. Änderungen dieser Erklärung

Diese Datenschutzerklärung wird erweitert, sobald neue Funktionen (z. B. Analytics, Nutzerkonten oder ein Scanner) hinzukommen.
